# Selenium Autotests (Chrome)

Автотесты для проекта находятся в этой директории.

## Что делает запуск

Скрипт:
- поднимает локальный HTTP-сервер для корня проекта;
- открывает новое окно Google Chrome;
- последовательно выполняет UI-тесты;
- выводит отчет `PASS/FAIL` в консоль.

## Подготовка

1. Убедитесь, что установлен Python 3.10+.
2. Убедитесь, что установлен Google Chrome.
3. Установите зависимости:

```bash
pip install -r autotests/requirements.txt
```

## Запуск

Из корня проекта:

```bash
python autotests/run_selenium_tests.py
```

Если все успешно, скрипт завершится с кодом `0`.
При наличии ошибок завершится с кодом `1`.
